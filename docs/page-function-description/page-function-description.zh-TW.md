## 第 1 步 - 輸入網址

可以選擇爬取整個網站或單一網址或自訂的網址 txt 檔。

<p align="center">
    <img src="../../Image/zh/001.png">
    <br />

</p>

## 第 2 步 - URL 清單

產生 URL 清單。

<p align="center">
    <img src="../../Image/zh/002.png">
    <br />
    右上角可以下載預設判斷影響 SEO 的網址檔案或全部網址檔案。
</p>

---

## 第 3 步 - 分析設定

進行分析的相關設定。

<p align="center">

</p>

## 第 4 步 - 執行中

<p align="center">

## 第 5 步 - 完成
